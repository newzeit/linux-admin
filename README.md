# Linux Admin Helper

Este repositorio contiene una colección de guías y notas útiles para administradores de sistemas Linux. Cada archivo está enfocado en una herramienta o tema específico, proporcionando ejemplos prácticos, comandos frecuentes y trucos para facilitar el trabajo diario.

## 📚 Guías Disponibles

- [AWK](awk.md) – Guía de uso de AWK para procesamiento de texto y generación de informes.
- [Git](git.md) – Comandos esenciales y flujos de trabajo con Git.
- [Linux Admin](linuxadmin.md) – Notas generales de administración Linux (usuarios, permisos, servicios, etc.).
- [Proxmox](proxmox.md) – Tips y comandos para gestionar Proxmox VE.
- [QEMU](qemu.md) – Uso de QEMU para virtualización y emulación.
- [Expresiones Regulares](regexp.md) – Sintaxis y ejemplos de expresiones regulares.
- [Zsh](zsh.md) – Configuración y uso avanzado de Zsh.

## 🛠️ Cómo usar

Cada archivo Markdown puede leerse directamente en GitHub o clonarse para uso local. Siéntete libre de contribuir con mejoras o nuevas guías mediante pull requests.
